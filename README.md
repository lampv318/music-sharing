# MY VIDEO PROJECT WITH BOOTSTRAP5

## Clone.

```
git@github.com:lampv318/my_video.git
```

## Required middleware.

#### ruby 3.0.3
#### Rails 7.0.3
#### Mysql 5.x, 8.x
#### NPM 6.x
#### BOOTSTRAP5

## Set up

```
bundle install --path .bundle
bundle exec rails db:migrate
```

## Run server

```
./bin/dev
```

## Run test

```
bundle exec rails test test/
```
